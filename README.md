RESUMEN DEL CÓDIGO

La Lista: Hay una lista llamada productos_cursoJs que son los productos (id, nombre y precio).

El Carrito: Existe un carrito llamado carrito_bootcampSamuel donde se guarda lo que eliges.

Agregar: La función agregarAlCarrito busca el producto y lo mete al carrito. Si ya estaba, solo le suma más cantidad.

La Cuenta: La función subTotal suma el precio de todo lo que metimos en el carrito.

Rebajas: La función aplicarDescuento busca "PROMO10" y te descuenta si cumples con el monto mínimo de compra.

El Recibo: La función resumen imprime en pantalla la lista de tus productos, el descuento que usaste y cuánto tienes que pagar al final.
